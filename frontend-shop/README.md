# MacMarket — Boutique (frontend-shop)

Application React (Vite + TypeScript strict) de la boutique client MacMarket : catalogue, panier, commande, assistant IA.

Voir le [README racine](../README.md) pour le lancement complet de la stack (`make up`, `make shop-run`) et les [ADR](../docs/adr/) pour les décisions d'architecture (notamment [ADR-0006](../docs/adr/0006-deux-frontends-react-separes.md) sur la séparation boutique/backoffice).

## Développement local

```bash
npm install
npm run dev      # http://localhost:5173
npm run build
npm test
```
