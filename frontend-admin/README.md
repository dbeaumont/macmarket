# MacMarket — Backoffice (frontend-admin)

Application React (Vite + TypeScript strict) du backoffice d'administration MacMarket : commandes, catalogue, clients, statistiques.

Voir le [README racine](../README.md) pour le lancement complet de la stack (`make up`, `make admin-run`) et les [ADR](../docs/adr/) pour les décisions d'architecture (notamment [ADR-0006](../docs/adr/0006-deux-frontends-react-separes.md) sur la séparation boutique/backoffice).

## Développement local

```bash
npm install
npm run dev      # http://localhost:5174
npm run build
npm test
```
